# Tie Professional Color Palette

## Official Brand Colors

### Primary: Navy Blue
**Hex:** `#2C3E50`  
**Usage:** Headers, navigation, primary text, structural elements, unselected states  
**Psychology:** Trust, professionalism, stability, authority

### Accent: Bright Yellow  
**Hex:** `#FFC107`  
**Usage:** CTAs, selected states, important highlights, "Publish" buttons  
**Psychology:** Energy, optimism, attention, action

### Secondary: Teal/Cyan
**Hex:** `#00BCD4` (main) / `#26C6DA` (light variant)  
**Usage:** Info badges, timestamps, capacity indicators, secondary actions, status displays  
**Psychology:** Innovation, clarity, modernity, trust

## Supporting Colors

### Blue (Tech Accent)
**Hex:** `#3B82F6`  
**Usage:** Gradients, links, informational elements  

### Grey Slate  
**Hex:** `#64748B`  
**Usage:** Secondary text, borders, subtle elements

### Success Green
**Hex:** `#10B981`  
**Usage:** Success messages, completed states ONLY

### Error Red
**Hex:** `#EF4444`  
**Usage:** Errors, warnings, destructive actions ONLY

## Design Principles

1. **Navy = Structure**: Use for all primary UI structure
2. **Yellow = Action**: Reserve for CTAs and selected states only  
3. **Teal = Information**: Use for all status, info, and timestamps
4. **Never mix yellow with teal** - keep them separate in purpose
